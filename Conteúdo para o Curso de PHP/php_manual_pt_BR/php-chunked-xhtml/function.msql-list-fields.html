msql_list_fields

(PHP 4, PHP 5 < 5.3.0)

msql_list_fields — Lista os campos de uma tabela

Descrição

msql_list_fields ( string $database , string $tablename [, resource $link_identifier ] ) : resource

msql_list_fields()obém informações sobre a tablename(nome da tabela) indicada. O conjunto de resultados retornado pode ser usado com quaisquer funções que obtém resultados de um conjunto de resultados como, por exemplo, msql_fetch_array().

Esta função retorna FALSE em caso de falha.

Para compatibilidade com versões anteriores, o apelido chamado msql_listfields() pode ser usado. Este, entretanto, esta obsoleto e seu uso não é recomendado.
