unlink

(PHP 4, PHP 5, PHP 7)

unlink — Apaga um arquivo

Descrição

unlink ( string $filename [, resource $context ] ) : int

Deleta filename. Similar a função do C no Unix unlink().

Parâmetros

filename

Caminho para o arquivo.

context

Nota: O suporte ao contexto foi adicionado no PHP 5.0.0. Para uma descrição de contextos, veja Streams.

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Changelog

Versão Descrição
5.0.0 No PHP 5.0.0 unlink() pode também ser usada com alguns URL wrappers. Consulte Protocolos e Wrappers suportados para uma listagem de wrappers que são suportados na unlink().
