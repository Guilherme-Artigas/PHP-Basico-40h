Exception::getCode

(PHP 5, PHP 7)

Exception::getCode — Obtém o código da exceção

Descrição

final public Exception::getCode ( void ) : mixed

Retorna o código da exceção.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

Retorna o código da Exceção como um integer da Exception mas possívelmente outros tipos descendentes de Exception (PDOException, por exemplo, retorna como string).

Exemplos

Exemplo #1 Exemplo da Exception::getCode()

<?php
try {
    throw new Exception("Some error message", 30);
} catch(Exception $e) {
    echo "The exception code is: " . $e->getCode();
}
?>

O exemplo acima irá imprimir algo similar à:

The exception code is: 30
