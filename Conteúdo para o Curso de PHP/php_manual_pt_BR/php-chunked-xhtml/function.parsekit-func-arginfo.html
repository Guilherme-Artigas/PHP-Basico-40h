parsekit_func_arginfo

(PECL parsekit >= 0.3.0)

parsekit_func_arginfo — Retorna informação de argumento(s) de uma função

Descrição

parsekit_func_arginfo ( mixed $function ) : array
Aviso

Esta função é EXPERIMENTAL. O comportamento, seu nome e documentação podem mudar sem aviso em futuras versões do PHP. Utilize por sua própria conta e risco.

Parâmetros

function

Uma string descrevendo uma função, ou um array descrevendo uma classe/método.

Valor Retornado

Retorna um array contendo informação dos argumentos.

Exemplos

Exemplo #1 parsekit_func_arginfo() example

<?php
function foo($bar, stdClass $baz, &$bomb, $bling = false) {
}

var_dump(parsekit_func_arginfo('foo'));
?>
