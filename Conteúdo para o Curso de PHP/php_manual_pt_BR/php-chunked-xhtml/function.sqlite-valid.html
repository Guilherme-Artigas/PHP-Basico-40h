(PHP 5 < 5.4.0)

sqlite_valid -- SQLiteResult::valid -- SQLiteUnbuffered::valid — Returns whether more rows are available

Descrição

sqlite_valid ( resource $result ) : bool

Estilo orientado à objeto (method):

SQLiteResult::valid ( void ) : bool
SQLiteUnbuffered::valid ( void ) : bool

Finds whether more rows are available from the given result handle.

Parâmetros

result

The SQLite result resource. This parameter is not required when using the object-oriented method.

Nota:

Esta função não pode ser utilizada com manipuladores de resultados sem buffer.

Valor Retornado

Returns TRUE if there are more rows available from the result handle, or FALSE otherwise.
