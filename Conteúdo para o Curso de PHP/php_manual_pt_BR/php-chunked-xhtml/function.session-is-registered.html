session_is_registered

(PHP 4, PHP 5 < 5.4.0)

session_is_registered — Verifica se uma variável global está registrada numa sessão

Descrição

session_is_registered ( string $name ) : bool

Verifica se uma variável global está registrada numa sessão.

Aviso

Esta função tornou-se OBSOLETA desde o PHP 5.3.0 e foi REMOVIDA desde o PHP 5.4.0.

Parâmetros

name

O nome da variável.

Valor Retornado

session_is_registered() retorna TRUE se há uma variável global com o nome name registrado na sessão atual, FALSE caso contrário.

Notas

Nota:

Se $_SESSION (ou $HTTP_SESSION_VARS para PHP 4.0.6 ou inferior) é usada, use isset() para checar se uma variável está registrada em $_SESSION.

Cuidado

Se você está usando $_SESSION (ou $HTTP_SESSION_VARS), não utilize session_register(), session_is_registered() e session_unregister().
