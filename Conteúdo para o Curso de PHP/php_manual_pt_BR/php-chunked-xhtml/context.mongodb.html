Opções de contexto do MongoDB

Opções de contexto do MongoDB — Lista de opções de contexto do MongoDB

Descrição

Opções de contexto do transportador mongodb://.

Opções

log_cmd_insert callable

Uma função callback chamada ao inserir um documento, veja a função log_cmd_insert().

log_cmd_delete callable

Uma função callback chamada ao remover um documento, veja a função log_cmd_delete().

log_cmd_update callable

Uma função callback chamada ao atualizar um documento, veja a função log_cmd_update().

log_write_batch callable

Uma função callback chamada ao executar uma gravação em lote, veja a função log_write_batch().

log_reply callable

Uma função callback chamada ao ler uma resposta do MongoDB, veja a função log_reply().

log_getmore callable

Uma função callback chamada ao recuperar mais resultados do cursor do MongoDB, veja a função log_getmore().

log_killcursor callable

Uma função callback chamada ao executar o opcode killcursor, veja a função log_killcursor().
