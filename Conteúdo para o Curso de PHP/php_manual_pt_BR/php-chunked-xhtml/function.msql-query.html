msql_query

(PHP 4, PHP 5 < 5.3.0)

msql_query — Envia uma consulta

Descrição

msql_query ( string $query [, resource $link_identifier ] ) : resource

msql_query() envia uma consulta para o banco de dados ativo no servidor indicado na conexão especificada.

Parâmetros

query

A query SQL.

link_identifier

A conexão mSQL. Se não for especificado, a ultima conexão aberta por msql_connect() é assumida. Se uma conexão assim não for encontrada, a função irá tentar establecer uma conexão como se a função msql_connect() fosse chamada, e irá usa-la.

Valor Retornado

Retorna um identificador de consulta em caso de sucesso, ou FALSE se ouver erro.

Exemplos

Exemplo #1 Exemplo msql_query()

<?php
$link = msql_connect("dbserver")
   or die("Não foi possível conectar ao servidor msql: " . msql_error()); 
msql_select_db("db", $link)
   or die("Não foi possível selecionar o banco de dados 'db': " . msql_error());

$result = msql_query("SELECT * FROM table WHERE id=1", $link);
if (!$result) {
   die("A consulta falhou: " . msql_error());
}

while ($row = msql_fetch_array($result)) {
    echo $row["id"];
}
?>
