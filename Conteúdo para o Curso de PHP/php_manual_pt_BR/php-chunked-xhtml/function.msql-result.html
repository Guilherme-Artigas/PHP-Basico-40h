msql_result

(PHP 4, PHP 5 < 5.3.0)

msql_result — Retorna dados do resultado

Description

msql_result ( resource $query_identifier , int $row [, mixed $field ] ) : string

Retorna o conteúdo de uma célula na linha e coluna especificadas de um conjunto de resultados.

msql_result() retorna o conteúdo de uma célula de um conjunto de resultados mSQL. O argumento field pode ser o nome do campo, seu índice numerico ou o nome da tabela ponto o nome do campo (tabela.campo). Se o campo tem um apelido ('select foo as bar from ...'), use o apelido ao invés do nome da coluna.

Quando estiver trabalhando com um conjunto de resultados grande, você deve considerar uma das funções que obtem o registro inteiro (especificadas abaixo). Já que estas funções retornam o conteúdo de multiplas células em uma chamada de função, elas são MUITO mais rápidas do que msql_result(). Também, note que especificar o índice numerico do argumento campo e muito mais rápido do que especificar o nome do campo ou tabela.campo.

Alternativas mais rápidas recomendadas: msql_fetch_row(), msql_fetch_array() e msql_fetch_object().

Para compatibilidade com versões anteriores, os apelidos chamados msql(), msql_tablename(), e msql_dbname() podem ser usados. Estes, estretanto, estão obsoletos e não são recomendados
