sybase_fetch_row

(PHP 4, PHP 5)

sybase_fetch_row — Obtém uma linha do resultado em uma matriz numérica

Descrição

sybase_fetch_row ( resource $result ) : array

sybase_fetch_row() obtém uma linha de dados do resultado associado ao identificador de resultado especificado.

As próximas chamadas a sybase_fetch_row() devem retornar a próxima linha no conjunto de resultados, ou FALSE se não houverem mais linhas.

Parâmetros

result

Valor Retornado

Retorna um array que corresponde a linha obtida, ou FALSE se houver mais linhas. Cada coluna de resultado é armazenada em um índice do array, iniciando em 0.

Tipos de dados
PHP Sybase
string VARCHAR, TEXT, CHAR, IMAGE, BINARY, VARBINARY, DATETIME
int NUMERIC (c/ ou s/ precision), DECIMAL (c/ o s/ precisão), INT, BIT, TINYINT, SMALLINT
float NUMERIC (c/ precisão), DECIMAL (c/ precisão), REAL, FLOAT, MONEY
NULL NULL
