O que mudou no PHP 5.0.x

PHP 5 e o integrado Zend Engine 2 tiveram grandes melhorias na performance e capacidades dos PHP's, mas também tiveram muito cuidado em quebrar a menor quantidade possível de códigos existentes. Portanto migrar seu código do PHP 4 para o 5 deve ser muito fácil. A maioria dos códigos PHP 4 existentes devem estar prontos para rodar sem alterações, mas você deve estar ciente sobre algumas diferenças e tomar cuidado de testar seu código antes de mudar as versões em seu ambiente de produção.
