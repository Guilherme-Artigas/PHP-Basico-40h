Introdução

Estas funções estão disponíveis somente quando o PHP estiver rodando como módulo do Apache.

Nota: A partir do PHP 4.3.2, a variável de ambiente PATH_TRANSLATED não é mais definida explicitamente na SAPI Apache 2, em contraste a esta situação, no Apache 1, onde é definida com o mesmo valor que a variável de ambiente SCRIPT_FILENAME quando não populada pelo Apache. Esta modificação foi feita para ser compátível com a especificação de CGI, que diz que PATH_TRANSLATED só deve existir se PATH_INFO for definida. Usuários de Apache 2 podem usar AcceptPathInfo = On dentro do httpd.conf para definir a variável de ambiente PATH_INFO.
