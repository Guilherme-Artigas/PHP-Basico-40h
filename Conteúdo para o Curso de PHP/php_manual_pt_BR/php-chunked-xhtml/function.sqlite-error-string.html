sqlite_error_string

(PHP 5 < 5.4.0, PECL sqlite >= 1.0.0)

sqlite_error_string — Retorna a descrição textual de um código de erro

Descrição

sqlite_error_string ( int $error_code ) : string

Retorna uma descrição legível de um código de erro (error_code) retornado por sqlite_last_error().
