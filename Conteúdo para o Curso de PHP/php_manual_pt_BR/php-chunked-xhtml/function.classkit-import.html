classkit_import

(PECL classkit >= 0.3)

classkit_import — Importa métodos definidos em outra classe

Descrição

classkit_import ( string $filename ) : array

Nota: Esta função não pode ser utilizada para manipular um método em uso (ou encadeado).

Aviso

Esta função é EXPERIMENTAL. O comportamento, seu nome e documentação podem mudar sem aviso em futuras versões do PHP. Utilize por sua própria conta e risco.

Parâmetros

filename

Nome do arquivo onde foi definido o método a ser importado

Valor Retornado

Array contendo os métodos importados

Exemplos

Exemplo #1 classkit_import() example

<?php
// file: newclass.php
class Example {
    function foo() {
        return "bar!\n";
    }
}
?>
<?php
// requires newclass.php (see above)
class Example {
    function foo() {
        return "foo!\n";
    }
}

$e = new Example();

// output original
echo $e->foo();

// import replacement method
classkit_import('newclass.php');

// output imported
echo $e->foo();

?>

O exemplo acima irá imprimir:

foo!
bar!
