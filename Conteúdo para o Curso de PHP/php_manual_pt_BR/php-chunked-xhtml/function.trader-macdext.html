trader_macdext

(PECL trader >= 0.2.0)

trader_macdext — MACD with controllable MA type

Descrição

trader_macdext ( array $real [, int $fastPeriod [, int $fastMAType [, int $slowPeriod [, int $slowMAType [, int $signalPeriod [, int $signalMAType ]]]]]] ) : array

Parâmetros

real

Array de valores reais.

fastPeriod

Número de períodos para o MA rápido. Valores válidos de 2 a 100000.

fastMAType

Tipo de Média Móvel para o MA rápido. As TRADER_MA_TYPE_* constantes devem ser utilizadas.

slowPeriod

Número de períodos para o MA lento. Valores válidos de 2 a 100000.

slowMAType

Tipo de Média Móvel para o MA lento. As TRADER_MA_TYPE_* constantes devem ser utilizadas.

signalPeriod

Amaciamento da linha de sinal (nb do período). Valores válidos de 1 a 100000.

signalMAType

Tipo de Média Móvel para a linha sinal. As TRADER_MA_TYPE_* constantes devem ser utilizadas.

Valor Retornado

Returns an array with calculated data or false on failure.
