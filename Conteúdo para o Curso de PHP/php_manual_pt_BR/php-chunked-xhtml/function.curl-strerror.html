curl_strerror

(PHP 5 >= 5.5.0, PHP 7)

curl_strerror — Retorna uma string que descreve o código de erro informado

Descrição

curl_strerror ( int $errornum ) : string

Retorna um texto descrevendo o código de erro informado como parâmetro.

Parâmetros

errornum

Uma das constantes de » códigos de erros do cURL.

Valor Retornado

Retorna a descrição do erro ou NULL se o código de erro é inválido.

Exemplos

Exemplo #1 curl_errno() example

<?php
// Cria um manipulador curl com um protocolo escrito de forma incorreta na URL
$ch = curl_init("htp://example.com/");

// Envia a requisição
curl_exec($ch);

// Verifica se houve erros e exibe a mensagem de erro
if($errno = curl_errno($ch)) {
    $error_message = curl_strerror($errno);
    echo "cURL error ({$errno}):\n {$error_message}";
}

// Fecha o manipulador
curl_close($ch);
?>

O exemplo acima irá imprimir:

cURL error (1):
 Unsupported protocol
