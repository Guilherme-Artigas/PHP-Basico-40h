gmp_binomial

(PHP 7 >= 7.3.0)

gmp_binomial — Calculates binomial coefficient

Descrição

gmp_binomial ( mixed $n , int $k ) : GMP

Calculates the binomial coefficient C(n, k).

Parâmetros

n

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

k

Valor Retornado

Returns the binomial coefficient C(n, k), ou FALSE em caso de falha.

Erros

Issues E_WARNING if k is negative.
