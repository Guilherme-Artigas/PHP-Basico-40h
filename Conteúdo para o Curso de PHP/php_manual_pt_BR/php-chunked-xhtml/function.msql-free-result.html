msql_free_result

(PHP 4, PHP 5 < 5.3.0)

msql_free_result — Libera o resultado de uma consulta da memória

Descrição

msql_free_result ( resource $result ) : bool

msql_free_result() libera a memória usada com query_identifier. Quando o PHP completa uma requisição, a memória é liberada automaticamente, assim você só precisa utilizar esta função quendo você quer ter certeza de que você não utiliza muita memória durante a execução do script.

Parâmetros

result

O resulatdo resource que esta sendo avaliado. Este resultado vem de uma chamada a msql_query().

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.
