sqlite_unbuffered_query

(PHP 5 < 5.4.0, PECL sqlite >= 1.0.0)

sqlite_unbuffered_query — Executa uma query que não guarda em buffer os resultados

Descrição

sqlite_unbuffered_query ( resource $dbhandle , string $query ) : resource
sqlite_unbuffered_query ( string $query , resource $dbhandle ) : resource

sqlite_unbuffered_query() é identica a sqlite_query() exceto que o resultado é retornado de uma maneira sequencial de acesso somente para frente, so pode ser usado para ler cada linha, uma após a outra.

Esta função é ideal para gerar coisas como tabelas em HTML aonde você só precisa processar uma linha de cada vez e não necessita de acesso aleatório aos dados.

Nota:

Funções como sqlite_seek(), sqlite_rewind(), sqlite_next(), sqlite_current() e sqlite_num_rows() não funcionam com conjunto de resultados retornados por sqlite_unbuffered_query().

Veja também sqlite_query().
