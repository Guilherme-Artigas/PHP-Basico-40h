Novos parâmetros

Algumas funções ganharam parâmetros opcionais novos no PHP 5.2.x:

Core do PHP:

  • htmlentities() - adicionado double_encode no PHP 5.2.3.
  • htmlspecialchars() - adicionado double_encode no PHP 5.2.3.
  • base64_decode() - adicionado strict
  • setcookie() - adicionado httponly
  • setrawcookie() - adicionado httponly
  • session_set_cookie_params() - adicionado httponly
  • memory_get_usage() - adicionado real_usage
  • get_loaded_extensions() - adicionado zend_extensions no PHP 5.2.4

curl:

  • curl_multi_info_read() - adicionado msgs_in_queue

datetime

  • date() - adicionado o caracterer de formatação "u" (millisegundos) no PHP 5.2.2

imap:

  • imap_open() - adicionado n_retries
  • imap_reopen() - adicionado n_retries

mbstring:

  • mb_strrpos() - adicionado offset
    Aviso

    O parãmetro offset foi colocado na posição onde parâmetro encoding estava. A compatibilidade com versões anteriores foi disponibilizada permitindo que o encoding seja especificado como terceiro parâmetro. Usar esse modo não é recomendado pois ele será removido em uma versão futura do PHP.

ming:

  • SWFMovie::streamMP3() - adicionado skip no PHP 5.2.1

openssl:

  • openssl_verify() - adicionado signature_algo

pgsql:

  • pg_escape_bytea() - adicionado connection
  • pg_escape_string() - adicionado connection

simplexml:

  • SimpleXMLElement::__construct() - adicionado is_prefix
  • SimpleXMLElement::attributes() - adicionado is_prefix
  • SimpleXMLElement::children() - adicionado is_prefix
  • simplexml_load_file() - adicionado is_prefix
  • simplexml_load_string() - adicionado is_prefix

spl:

  • array iterator_to_array(Traversable it [, bool use_keys = true]) - adicionado use_keys in PHP 5.2.1

xmlreader:

  • XMLReader::open() - adicionado encoding e options
  • XMLReader::XML() - adicionado encoding e options

XMLWriter:

  • xmlwriter_write_element() - o parâmetro content tornou-se opcional no PHP 5.2.3
  • xmlwriter_write_element_ns() - o parâmetro content tornou-se opcional no PHP 5.2.3
