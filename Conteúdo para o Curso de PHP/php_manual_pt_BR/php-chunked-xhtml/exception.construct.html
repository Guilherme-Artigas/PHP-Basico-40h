Exception::__construct

(PHP 5, PHP 7)

Exception::__construct — Construtor da exceção

Descrição

public Exception::__construct ([ string $message = NULL [, int $code [, Throwable $previous = NULL ]]] )

Construtor da exceção.

Parâmetros

message

A mensagem da exceção a ser disparada.

code

O código da exceção.

previous

A exceção anteriormente usada para encadeamento de exceções.

Changelog

Versão Descrição
7.0.0 O parâmetro previous agora é do tipo Throwable.
5.3.0 O parâmetro excecao_anterior foi adicionado.

Notas

Nota:

O parâmetro message NÃO é binary safe.
