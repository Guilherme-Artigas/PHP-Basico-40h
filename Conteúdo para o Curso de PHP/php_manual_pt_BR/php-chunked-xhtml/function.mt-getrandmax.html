mt_getrandmax

(PHP 4, PHP 5, PHP 7)

mt_getrandmax — Retorna o maior valor aletório possível

Descrição

mt_getrandmax ( void ) : int

Retorna o maior valor que pode ser obtido numa chamada a mt_rand().

Valor Retornado

Retorna o maior valor possível de uma chamada a mt_rand() sem argumentos, que o máximo valor que pode ser utilizado no parâmetro max sem o resultado ser escalonado (perdendo assim aleatoriedade).

Exemplos

Exemplo #1 Calcula um número aleatório de ponto flutuante

<?php
function randomFloat($min = 0, $max = 1) {
    return $min + mt_rand() / mt_getrandmax() * ($max - $min);
}

var_dump(randomFloat());
var_dump(randomFloat(2, 20));
?>

O exemplo acima irá imprimir algo similar à:

float(0.91601131712832)
float(16.511210331931)
