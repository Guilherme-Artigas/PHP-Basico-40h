crack_closedict

(PECL crack >= 0.1)

crack_closedict — Fecha um dicionário CrackLib aberto

Descrição

crack_closedict ([ resource $dictionary ] ) : bool

crack_closedict() fecha o identificador de dictionary indicado.

Aviso

Esta função é EXPERIMENTAL. O comportamento, seu nome e documentação podem mudar sem aviso em futuras versões do PHP. Utilize por sua própria conta e risco.

Parâmetros

dictionary

O dicinário para fechar. Se não for especificado, o diretório atual é fechado.

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.
