mssql_num_fields

(PHP 4, PHP 5, PECL odbtp >= 1.1.1)

mssql_num_fields — Retorna o número de campos em um resultado

Descrição

mssql_num_fields ( resource $result ) : int

mssql_num_fields() retorna o número de campos em um conjunto de resultados.

Parâmetros

result

O resource de resultado que está sendo avaliado. Este resultado vem da chamada a mssql_query().

Valor Retornado

Retorna o número de campos, como um inteiro.
