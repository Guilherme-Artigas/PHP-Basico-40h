msql_field_len

(PHP 4, PHP 5 < 5.3.0)

msql_field_len — Obtém o tamanho do campo

Descrição

msql_field_len ( resource $result , int $field_offset ) : int

msql_field_len() retorna o tamanho do campo especificado.

Parâmetros

result

O resulatdo resource que esta sendo avaliado. Este resultado vem de uma chamada a msql_query().

field_offset

O número índice do campor. field_offset começa com 1.

Valor Retornado

Retorna o tamanho do campo especificado ou FALSE em erro.
