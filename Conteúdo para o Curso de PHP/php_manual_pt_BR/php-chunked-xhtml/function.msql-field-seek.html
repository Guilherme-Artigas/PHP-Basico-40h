msql_field_seek

(PHP 4, PHP 5 < 5.3.0)

msql_field_seek — Estabelece o índice do campo

Descrição

msql_field_seek ( resource $result , int $field_offset ) : bool

Move o índice de campos para o campo especificado. Na próxima chamada a msql_fetch_field() que não inclui um índice de campo, este campo será retornado.

Parâmetros

result

O resulatdo resource que esta sendo avaliado. Este resultado vem de uma chamada a msql_query().

field_offset

O número índice do campor. field_offset começa com 1.

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.
