MongoClient::connect

(PECL mongo >=1.3.0)

MongoClient::connect — Connects to a database server

A extensão que define esse método está obsoleta. Alternativamente a extensão MongoDB deve ser utilizada. Não há um equivalente deste método na nova extensão.

Descrição

public MongoClient::connect ( void ) : bool

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

If the connection was successful.

Erros

Throws MongoConnectionException if it fails to connect to the database.
