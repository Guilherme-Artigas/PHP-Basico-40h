variant_and

(PHP 5, PHP 7)

variant_and — Performs a bitwise AND operation between two variants

Descrição

variant_and ( mixed $left , mixed $right ) : variant

Performs a bitwise AND operation. Note that this is slightly different from a regular AND operation.

Parâmetros

left

The left operand.

right

The right operand.

Nota:

Assim como todas as funções aritméticas, os parâmetros para esta função podem ser qualquer um tipo nativo do PHP (integer, string, ponto flutuante, booleano ou NULL), ou uma instancia de uma classe COM, VARIANT ou DOTNET. Tipos nativos do PHP serão convertidos para variants usando as mesmas regras como mencionadas no construtor da classe variant. Objetos COM e DOTNET terão o valor de suas propriedades padrão obtidos e usados como um valor variant.

As funções aritméticas variant são wrappers para funções chamadas similarmente na biblioteca COM; para mais informação sobre estas funções, consulte o MSDN. As funções PHP são nomeadas ligeiramente diferentes; por exemplo variant_add() no PHP corresponde à VarAdd() na documentação MSDN.
