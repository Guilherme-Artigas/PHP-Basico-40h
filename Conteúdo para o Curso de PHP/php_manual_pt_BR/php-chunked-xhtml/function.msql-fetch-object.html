msql_fetch_object

(PHP 4, PHP 5 < 5.3.0)

msql_fetch_object — Obtem uma linha como um objeto

Descrição

msql_fetch_object ( int $query_identifier ) : int

Retorna um objeto cuja as propriedades correspondem a linha obtida, ou FALSE se não houverem mais linhas.

msql_fetch_object() é similar a msql_fetch_array(), com uma diferença - um objeto é retorado em vez de uma matriz, isto indica que você somente pode acessar os dados pelos nomes dos campos e não por seus índices(números não são nomes de propriedade válidos).

Em termos de velocidade esta função é identica a msql_fetch_array(), e quase tão rapida quanto msql_fetch_row() (a diferença é insignificante).

Veja também msql_fetch_array() e msql_fetch_row().
