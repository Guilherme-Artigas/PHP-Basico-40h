win32_ps_stat_mem

(PECL win32ps >= 1.0.1)

win32_ps_stat_mem — Estatística de utilização de memória

Descrição

win32_ps_stat_mem ( void ) : array

Obtem estatísticas sobre utilização de memória no sistema.

Valor Retornado

Retorna FALSE em caso de falha, ou um array com as seguintes informações:

load

A percentagem de carga da memória física.

unit

Isso sempre é 1024, e indica que os seguintes valores tem como unidade 1024 bytes.

total_phys

A quantidade total de memória física.

avail_phys

A quantidade disponível de memória física.

total_pagefile

A quantidade total de memória paginável (memória física + swap).

avail_pagefile

A quantidade disponível de memória paginável (memória física + swap).

total_virtual

A quantidade total de memória virtual para um processo.

avail_virtual

A quantidade disponível de memória virtual para um processo.
