bbcode_parse

(PECL bbcode >= 0.9.0)

bbcode_parse — Analisa uma string seguindo um dado conjunto de regras

Descrição

bbcode_parse ( resource $bbcode_container , string $to_parse ) : string

Esta função analisa a string to_parse seguindo as regras em no bbcode_container criado por bbcode_create()

Parâmetros

bbcode_container

Resource BBCode_Container retornado pela bbcode_create().

to_parse

A string que precisa analisar.

Valor Retornado

Retorna a string interpretada, ou FALSE em caso de falha.
