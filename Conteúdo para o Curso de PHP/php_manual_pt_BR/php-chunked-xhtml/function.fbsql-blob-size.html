fbsql_blob_size

(PHP 4 >= 4.2.0, PHP 5 < 5.3.0)

fbsql_blob_size — Get the size of a BLOB

Descrição

fbsql_blob_size ( string $blob_handle [, resource $link_identifier ] ) : int

Returns the size of the given BLOB.

Parâmetros

blob_handle

A BLOB handle, returned by fbsql_create_blob().

link_identifier

Um identificador de link FrontBase retornado porfbsql_connect() ou fbsql_pconnect().

Se opcional e não especificado a função tentará localizar um link aberto ao um servidor FrontBase e se nenhum link for encontrado será tentado criar um como se fbsql_connect() fosse chamado sem argumentos.

Valor Retornado

Returns the BLOB size as an integer, or FALSE on error.
