Instalação

Instalando do PECL

  1. Você pode fazer download da php_win32ps.dll no pecl4win.php.net.

  2. Copie a php_win32ps.dll para seu extension_dir.

  3. Carrege a extensão, adicionando essa linha no arquivo php.ini 

    extension=php_win32ps.dll
