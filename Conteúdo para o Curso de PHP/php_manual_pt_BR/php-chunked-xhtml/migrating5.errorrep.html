Relatório de Erros

A partir do PHP 5 a constante de relatório de erros E_STRICT está disponível, com o valor 2048. Quando habilitado, serão exibidas mensagens para alertar sobre uso de código que está obsoleto ou que pode não estar seguro no futuro.

Nota: E_ALL não inclui E_STRICT, portanto não está habilitado por padrão. Você deve setar explicitamente o nível do relatório de errros para incluir o E_STRICT e visualizar as mensagens.
