fam_pending

(PHP 5 < 5.1.0)

fam_pending — Verifica por eventos FAM pendentes

Descrição

fam_pending ( resource $fam ) : int

Verifica por eventos FAM pendentes.

Parâmetros

fam

Um resource representando uma conexão para um serviço FAM retornado pela fam_open()

Valor Retornado

Retorna um valor diferente de zero se eventos estão disponíveis para serem obtidos usando fam_next_event(), caso contrário zero.
