Safe Mode

Índice

  • Segurança e o modo seguro (safe mode)
  • Funções restritas/desabilitadas pelo "safe mode"

O modo seguro (safe mode) do PHP tem como objetivo resolver um dos problemas de segurança encontrado em servidores compartilhados. Arquiteturalmente falando é incorreto tentar resolver este problema no nível do PHP, porém, como as alternativas nos servidores web e em nível de sistema operacional não são muito realistas, muitas pessoas, especialmente os provedores, utilizam o "safe mode" no momento.

Aviso

Esta funcionalidade tornou-se OBSOLETA desde o PHP 5.3.0 e foi REMOVIDA desde o PHP 5.4.0.

Log de atualização safe mode
Versão Descrição
5.4.0 Removido do PHP, podendo causar um erro fatal E_CORE_ERROR quando estiver habilitado.
5.3.0 Descontinuado, o aviso E_DEPRECATED será disparado.
