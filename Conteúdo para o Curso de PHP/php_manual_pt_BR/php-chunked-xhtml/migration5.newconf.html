Novas Diretivas

Existem algumas diretivas novas no php.ini introduzidas no PHP 5. Aqui está uma lista delas:

  • mail.force_extra_parameters - Força a adição de parâmetros específicos para serem passados como parâmetros adicionais no binário sendmail. Estes parâmetros sempre serão substituídos pelo valor do quinto parâmetro da função mail(), mesmo em modo de segurança.
  • register_long_arrays - autoriza/desautoriza o PHP a registrar as obsoletas $HTTP_*_VARS
  • session.hash_function - seleciona uma função de hash (MD5 or SHA-1)
  • session.hash_bits_per_character - define quantos bits são armazenados em cada caracter quando converte o valor do hash binário para algo legível (de 4 a 6)
  • zend.ze1_compatibility_mode - Habilita o modo de compatibilidade com o Zend Engine 1 (PHP 4)
