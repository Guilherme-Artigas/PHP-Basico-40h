Novos métodos

Novos métodos foram introduzidos no PHP 5.2.0:

dom:

  • DOMDocument::registerNodeClass() - Registra uma classe estendida usada para criar um tipo de nó base
  • DOMElement::setIDAttribute() - Declara o atributo especificado pelo nome para ser do tipo ID
  • DOMElement::setIDAttributeNode() - Declara o atributo especificado pelo nó para ser do tipo ID
  • DOMElement::setIDAttributeNS() - Declara o atributo especificado pelo nome local e pelo URI do namespace para ser do tipo ID
  • DOMNode::C14N()([bool exclusive [, bool with_comments [, array xpath [, array ns_prefixes]]]]) - Canoniza os nós para uma string
  • DOMNode::C14NFile()(string uri [, bool exclusive [, bool with_comments [, array xpath [, array ns_prefixes]]]]) - Canoniza os nós para um arquivo
  • DOMNode::getNodePath()() - Obtém um xpath para um nó

soap:

  • SoapServer::setObject()(object obj) - Define o objeto que seja manipulado pelas requisições SOAP

spl:

  • int ArrayObject::asort()(void) - Organiza as entradas pelos valores
  • int ArrayObject::ksort()(void) - Organiza as entradas pelas chaves
  • int ArrayObject::natcasesort()(void) - Organiza as entradas pelas chaves usando o algoritmo case insensitive de "ordem natural"
  • int ArrayObject::natsort()(void) - Organiza as entradas por valores usando o algoritmo de "ordem natural".
  • int ArrayObject::uasort()(callback cmp_function) - Organiza as entradas por valores usando uma função definida pelo usuário
  • int ArrayObject::uksort()(callback cmp_function) - Organiza as entradas por chaves usando uma função definida pelo usuário.
  • ArrayIterator AppendIterator::getArrayIterator()() - Obem acesso ao ArrayIterator interno
  • int AppendIterator::getIteratorIndex()() - Obtém o índice do iterator
  • bool CachingIterator::getCache()() - Retorna o cache
  • int CachingIterator::getFlags()() - Retorna o sinalizador interno
  • bool CachingIterator::offsetExists()(mixed index) - Retorna TRUE se o índice solicitado existir
  • string CachingIterator::offsetGet()(mixed index) - Retorna o cache interno de usado
  • void CachingIterator::offsetSet()(mixed index, mixed newval) - Define o índice dado no cache
  • void CachingIterator::offsetUnset()(mixed index) - Remove o índice dado no cache
  • void CachingIterator::setFlags()() - Define o sinalizador interno
  • array("delimiter" =>, "enclosure" =>) SplFileObject::getCsvControl()(void) - Obtém o delimitador e o enclosure usado na função fgetcsv()
  • void SplFileObject::setCsvControl()([string delimiter = ',' [, string enclosure = '"']]) - Define o delimitador e o enclosure usado na função fgetcsv()

Tidy

  • tidyNode tidyNode::getParent()() - Retorna o nó pai do nó atual (Adicionado no PHP 5.2.2)

XMLReader

  • boolean XMLReader::setSchema() - Usa o esquema W3C XSD para validar o documento da forma como ele é processado. A Ativação somente é possível antes do primeiro XMLReader::read()

zip:

  • ZipArchive::addEmptyDir()() - Cria um diretório vazio no arquivo
