bcdiv

(PHP 4, PHP 5, PHP 7)

bcdiv — Dividir dois números de precisão arbitrária

Descrição

bcdiv ( string $left_operand , string $right_operand [, int $scale = 0 ] ) : string

Divide o left_operand pelo right_operand.

Parâmetros

left_operand

O operando da esquerda, como uma string.

right_operand

O operando da direita, como uma string.

scale

Este parâmetro opcional é utilidazado para configurar o número de dígitos na parte fracionária do resultado. Se omitido será utilizada a escala padrão globalmente configurada com a função bcscale(), com padrão de fábrica em 0 se não configurado.

Valor Retornado

Retorna o resultado da divisão como uma string, ou NULL se right_operand é 0.

Exemplos

Exemplo #1 Exemplo da bcdiv()

<?php

echo bcdiv('105', '6.55957', 3);  // 16.007

?>
