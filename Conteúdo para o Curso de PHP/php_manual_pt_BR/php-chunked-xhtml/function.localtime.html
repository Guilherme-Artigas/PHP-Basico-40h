localtime

(PHP 4, PHP 5, PHP 7)

localtime — Obtém a hora local

Descrição

localtime ([ int $timestamp [, bool $is_associative ]] ) : array

A função localtime() retorna um array idêntico a estrutura retornada pela chamada de uma função em C.

Parâmetros

timestamp

O parâmetro opcional timestamp é um integer Unix timestamp cujo padrão é a hora local se timestamp não for informado. Em outras palavras, o padrão é o valor da função time().

is_associative

Se definido para FALSE ou não fornecido, um array normal é retornado, um array numericamente indexado. Se o argumento é definido para TRUE, localtime() retornará um array associativo contendo todos os diferentes elementos da estrutura retornados pela chamada da função C para o localtime. Os nomes das chaves do array associativo são as seguintes:

  • "tm_sec" - segundos, 0 a 59
  • "tm_min" - minutos, 0 a 59
  • "tm_hour" - horas, 0 a 23
  • "tm_mday" - dia do mês, 1 a 31
  • "tm_mon" - mês do ano, 0 (Jan) a 11 (Dec)
  • "tm_year" - anos desde 1900
  • "tm_wday" - dia da semana, 0 (Sun) a 6 (Sat)
  • "tm_yday" - dia do ano, 0 a 365
  • "tm_isdst" - Se está em horário de verão Positivo se sim, 0 se não, negative se é desconhecido.

Erros

Todas as chamadas a funções de data/hora gerarão um E_NOTICE se o fuso horário não for válido, e/ou uma mensagem E_STRICT ou E_WARNING se utilizar as configurações do sistema ou a variável de ambiente TZ. Veja também date_default_timezone_set()

Exemplos

Exemplo #1 Exemplo da função localtime()

<?php
$localtime = localtime();
$localtime_assoc = localtime(time(), true);
print_r($localtime);
print_r($localtime_assoc);
?>

O exemplo acima irá imprimir algo similar à:
