Configurações em Execução

O comportamento dessas funções podem ser modificado pelas configurações do php.ini.

Opções de Configuração do Tidy
Nome Padrão Modificável Changelog
tidy.default_config "" PHP_INI_SYSTEM Disponível desde o PHP 5.0.0.
tidy.clean_output "0" PHP_INI_USER PHP_INI_PERDIR no PHP 5. Disponível desde o PHP 5.0.0.
Para mais detalhes e definições dos modos PHP_INI_*, veja Onde uma configuração deve ser definida.

Uma rápida explicação das diretivas de configuração.

tidy.default_config string

Caminho padrão para o arquivo de configuração do tidy.

tidy.clean_output boolean

Torna on/off a saída reparada pelo Tidy.

Aviso

Não torne on tidy.clean_output se você está gerando conteúdo não-html como imagens dinâmicas.
