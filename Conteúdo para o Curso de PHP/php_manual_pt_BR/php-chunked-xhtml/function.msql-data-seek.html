msql_data_seek

(PHP 4, PHP 5 < 5.3.0)

msql_data_seek — Move o ponteiro interno dos registros

Descrição

msql_data_seek ( resource $result , int $row_number ) : bool

msql_data_seek() move o ponteiro interno de registro para o conjunto de resultados identificado por query_identifier para o número do registro indicado. A próxima chamada a msql_fetch_row() irá retornar este registro.

Parâmetros

result

O resulatdo resource que esta sendo avaliado. Este resultado vem de uma chamada a msql_query().

row_number

A linha desejada.

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.
