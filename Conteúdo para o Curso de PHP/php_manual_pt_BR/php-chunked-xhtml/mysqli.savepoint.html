mysqli::savepoint

mysqli_savepoint

(PHP 5 >= 5.5.0, PHP 7)

mysqli::savepoint -- mysqli_savepoint — Set a named transaction savepoint

Descrição

Estilo orientado à objeto

public mysqli::savepoint ( string $name ) : bool

Estilo procedural:

mysqli_savepoint ( mysqli $link , string $name ) : bool

Aviso

Esta função não está documentada; somente a lista de argumentos está disponível.

Parâmetros

link

Somente no estilo procedural: Um recurso link retornado por mysqli_connect() ou mysqli_init()

name

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.
