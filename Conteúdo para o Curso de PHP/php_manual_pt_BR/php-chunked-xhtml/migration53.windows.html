Mudanças feitas para suporte ao Windows

Mudanças para a versão do Windows:

  • O versão mínima do Windows agora é o Windows XP SP3. Windows 98, ME, 2000 e NT não são mais suportados.
  • Os binários do Windows são construídos para i586 e mais novos. i386 e i486 não são suportados.
  • Agora existe um suporte experimental para versões x64 do PHP para Windows.
  • Agora existe suporte para o compilador do Visual C++ 9 (VC9), usando Visual Studio 2008. Snapshots e releases serão disponibilizadas para VC9. Binários antigos usando VC6 ainda são suportados e disponibilizados em linha com o VC9.
  • A biblioteca PDO_OCI php_pdo_oci8.dll (para uso com bibliotecas cliente do Oracle versão 8) não está mais sendo construída. Em vez disso, use php_pdo_oci.dll (note que não tem '8') com bibliotecas cliente do Oracle 10 ou 11. Conexões para outras versões do banco de dados ainda são suportadas.
  • Para a extensão OCI8, uma nova biblioteca php_oci8_11g.dll está disponível em adição à php_oci8.dll. Só uma pode ser ativada por vez. Use php_oci8.dll com bibliotecas cliente do Oracle 10.2. Use php_oci8_11g.dll com bibliotecas cliente do Oracle 11. Conexões para outras versões do banco de dados ainda são suportadas.

Suporte ao Windows foi adicionado para as seguintes funções:

  • checkdnsrr()
  • dns_get_record()
  • fnmatch()
  • getmxrr()
  • getopt()
  • imagecolorclosesthwb()
  • inet_ntop()
  • inet_pton()
  • link()
  • linkinfo()
  • mcrypt_create_iv()
  • readlink()
  • socket_create_pair() - Esta função já estava disponível no Windows, mas foi desativada no PHP 4.3.0 devido à um bug.
  • stream_socket_pair()
  • symlink()
  • time_nanosleep()
  • time_sleep_until()

Outras modifições:

  • Melhoramos a portabilidade das funções stat(), touch(), filemtime(), filesize(), e outras funções relacionadas (100% portável para o dado disponível).
  • Agora é possível criar hard link no Windows usando a função link(), e link simbólico usando a função symlink(). Hard links estão disponívies no Windows 2000, e links simbólicos no Windows Vista.
  • The Windows version of PHP now exposes a set of constants prefixed PHP_WINDOWS_*. A list of these constants and their usage can be found at Constantes pré-definidas.
Aviso

Suporte para o módulo ISAPI foi removido. Use o módulo aprimorado do FastCGI SAPI no lugar.

Nota: Um novo site dedicado para o PHP no Windows está disponível, incluindo downloads, release candidates, e snapshots em vários modos (thread-safe/not-thread-safe, VC6/VC9, x86/x64). A URL deste site é » https://windows.php.net/.
