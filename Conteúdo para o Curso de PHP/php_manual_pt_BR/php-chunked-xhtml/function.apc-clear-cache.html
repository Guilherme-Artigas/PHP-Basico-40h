apc_clear_cache

(PECL apc >= 2.0.0)

apc_clear_cache — Limpa o cache do APC.

Descrição

apc_clear_cache ([ string $cache_type = "" ] ) : bool

Limpa o cache do usuário/sistema.

Parâmetros

cache_type

Se cache_type for configurado com "user", o cache do usuário será limpado; caso contrário, o cache do sistema (arquivos cacheados) será limpo.

Valor Retornado

Sempre retornará TRUE
