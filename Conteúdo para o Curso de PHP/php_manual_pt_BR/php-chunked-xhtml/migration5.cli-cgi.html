CLI e CGI

No PHP 5 teve algumas mudanças nos nomes de arquivos CLI e CGI. No PHP 5, a versão CGI foi renomeada para php-cgi.exe (anteriormente era php.exe) e a versão CLI agora está no diretório principal (anteriormente estava em cli/php.exe).

No PHP 5 foi adicionado um novo modo: php-win.exe. Ele é igual à versão CLI, exceto que o php-win não exibe nenhum e não provê um console (não há uma "janela dos" na tela). Este comportamento é similar ao php-gtk.

No PHP 5, a versão CLI será irá popular as variáveis globais $argv e $argc independente de qualquer diretiva setada no php.ini. Mesmo tendo register_argc_argv setado para off não afetará a CLI.
