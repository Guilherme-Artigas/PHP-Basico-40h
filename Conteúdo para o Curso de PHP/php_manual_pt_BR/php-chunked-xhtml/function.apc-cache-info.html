apc_cache_info

(PECL apc >= 2.0.0)

apc_cache_info — Recupera informações cacheadas do armazenamento do APC

Descrição

apc_cache_info ([ string $cache_type = "" [, bool $limited = false ]] ) : array

Recupera informações cacheadas do armazenamento do APC.

Parâmetros

cache_type

Se cache_type for configurado com "user", informações sobre o cache do usuário serão retornadas.

Se cache_type for configurado com "filehits", informação sobre quais arquivos estão sendo servidos do cache de bytecode para a requisição atual serão retornadas. Este recurso precisa ser habilitado na compilação usando --enable-filehits.

Se cache_type for inválido ou não especificado, informação sobre o sistema de cache (arquivos em cache) será retornada.

limited

Se limited for configurado como TRUE, o valor retornado excluirá a lista individual de entradas do cache. Isso é útil quando estiver tentando otimizar chamadas para obter estatísticas.

Valor Retornado

Array dos dados em cache (e meta-data) ou FALSE em caso de falha

Nota: apc_cache_info() lançará um aviso se for incapaz de retorna os dados em cache. Isso normalmente acontece quando o APC não estiver habilitado.

Changelog

Versão Descrição
3.0.11 O parâmetro limited foi introduzido.
3.0.16 A opção "filehits" para o parâmetro cache_type foi introduzida.

Exemplos

Exemplo #1 Um exemplo de apc_cache_info()

<?php
print_r(apc_cache_info());
?>

O exemplo acima irá imprimir algo similar à:

Array
(
    [num_slots] => 2000
    [ttl] => 0
    [num_hits] => 9
    [num_misses] => 3
    [start_time] => 1123958803
    [cache_list] => Array
        (
            [0] => Array
                (
                    [filename] => /path/to/apc_test.php
                    [device] => 29954
                    [inode] => 1130511
                    [type] => file
                    [num_hits] => 1
                    [mtime] => 1123960686
                    [creation_time] => 1123960696
                    [deletion_time] => 0
                    [access_time] => 1123962864
                    [ref_count] => 1
                    [mem_size] => 677
                )
            [1] => Array (...iterates for each cached file)
)
