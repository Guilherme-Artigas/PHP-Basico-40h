array_pop

(PHP 4, PHP 5, PHP 7)

array_pop — Extrai um elemento do final do array

Descrição

array_pop ( array &$array ) : mixed

array_pop() extrai e retorna o último elemento de array, diminuindo array em um elemento.

Nota: Esta função irá resetar() o ponteiro do array depois do uso.

Parâmetros

array

O array para obter os valores.

Valor Retornado

Retorna o último valor de array. Se array é vazio (ou ele não é um array), NULL será retornado.

Erros

Esta função irá lançar um erro nível E_WARNING se chamada com um parâmetro não array.

Exemplos

Exemplo #1 Exemplo da array_pop()

<?php
$cesta = array("laranja", "banana", "melancia", "morango");
$fruta = array_pop($cesta);
print_r($cesta);
?>

After this, $stack will have only 3 elements: 

Array
(
    [0] => laranja
    [1] => banana
    [2] => melancia
)

e morango será passado para $fruta.
