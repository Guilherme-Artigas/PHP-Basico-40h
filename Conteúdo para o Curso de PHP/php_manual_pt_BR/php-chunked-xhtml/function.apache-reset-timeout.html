apache_reset_timeout

(PHP 5 >= 5.1.0, PHP 7)

apache_reset_timeout — Reinicializa o timer de escrita do Apache

Descrição

apache_reset_timeout ( void ) : bool

A função apache_reset_timeout() reinicializá o temporizador de escrita do Apache, que tem o valor padrão de 300 segundo. Com set_time_limit(0); ignore_user_abort(true) e chamadas periódicas da função apache_reset_timeout(), o Apache pode, teoricamente, executar para sempre.

Essa função requer Apache 1.

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Notas

Nota: Está função está desabilitada quando o PHP é rodado sob o safe mode.
