getrandmax

(PHP 4, PHP 5, PHP 7)

getrandmax — Retorna o maior valor aletório possível

Descrição

getrandmax ( void ) : int

Retorna o maior valor que pode ser gerado em uma chamada a função rand().

Valor Retornado

O maior valor aleatório possível retornado pela função rand()
