Constantes pré-definidas

As contantes abaixo são definidas por esta extensão e somente estarão disponíveis quando a extensão foi compilada com o PHP ou carregada dinamicamente durante a execução.

ZipArchive utiliza constantes de classe. Há três tipos de constantes: Flags (prefixadas com FL_), erros (prefixados com ER_) e modo (sem prefixo).

ZipArchive::CREATE (integer)
Cria o arquivo se ele não existir.
ZIPARCHIVE::OVERWRITE (integer)
Se o arquivo existir, ignora o conteúdo atual. Em outras palavras, funciona da mesma fora que um arquivo vazio
ZipArchive::EXCL (integer)
Emite erro se o arquivo já existir.
ZipArchive::RDONLY (integer)
Open archive in read only mode. Available as of PHP 7.4.3 and PECL zip 1.17.1, respectively, if built against libzip ≥ 1.0.0.
ZipArchive::CHECKCONS (integer)
Executa uma verificação adicional de consistência no arquivo, e emite erro se ele falhar.
ZipArchive::FL_NOCASE (integer)
Ignora maiúsculo e minúsculo no nome.
ZipArchive::FL_NODIR (integer)
Ignora componentes de diretório.
ZipArchive::FL_COMPRESSED (integer)
Lê a informação comprimida.
ZipArchive::FL_UNCHANGED (integer)
Usa os dados originais, ignorando mudanças.
ZipArchive::FL_RECOMPRESS (integer)
Forçar a recompressão de dados. disponível a partir do PHP 8.0.0 e PECL zip 1.18.0.
ZipArchive::FL_ENCRYPTED (integer)
Ler dados criptografados (implica FL_COMPRESSED). Disponível a partir do PHP 8.0.0 e PECL zip 1.18.0.
ZipArchive::FL_OVERWRITE (integer)
Se o arquivo com nome existir, substitua-o (replace). Disponível a partir do PHP 8.0.0 e PECL zip 1.18.0.
ZipArchive::FL_LOCAL (integer)
No cabeçalho local. Disponível a partir do PHP 8.0.0 e PECL zip 1.18.0.
ZipArchive::ZIP_FL_CENTRAL (integer)
No diretório central. Disponível a partir do PHP 8.0.0 e PECL zip 1.18.0.
ZipArchive::FL_ENC_GUESS (integer)
Melhor para descompactar ou armazenar. Disponível no PHP 7.0.8.
ZipArchive::FL_ENC_RAW (integer)
Obtém sem modificações. Disponível no of PHP 7.0.8.
ZipArchive::FL_ENC_STRICT (integer)
Segue a especificação estritamente. Disponível no PHP 7.0.8.
ZipArchive::FL_ENC_UTF_8 (integer)
Strings codificadas como UTF-8 encoded. Disponível no PHP 7.0.8.
ZipArchive::FL_ENC_CP437 (integer)
Strings codificadas como CP437. Disponível no PHP 7.0.8.
ZipArchive::CM_DEFAULT (integer)
better of deflate or store.
ZipArchive::CM_STORE (integer)
stored (uncompressed).
ZipArchive::CM_SHRINK (integer)
shrunk
ZipArchive::CM_REDUCE_1 (integer)
Compressão com fator 1
ZipArchive::CM_REDUCE_2 (integer)
Compressão com fator 2
ZipArchive::CM_REDUCE_3 (integer)
Compressão com fator 3
ZipArchive::CM_REDUCE_4 (integer)
Compressão com fator 4
ZipArchive::CM_IMPLODE (integer)
Implode
ZipArchive::CM_DEFLATE (integer)
descompactado
ZipArchive::CM_DEFLATE64 (integer)
deflate64
ZipArchive::CM_PKWARE_IMPLODE (integer)
Compactação PKWARE
ZipArchive::CM_BZIP2 (integer)
Algoritmo BZIP2
ZipArchive::CM_LZMA (integer)
Algoritmo LZMA
ZipArchive::CM_LZMA2 (integer)
Algoritmo LZMA2. Disponível a partir do PHP 7.4.3 e PECL zip 1.16.0, respectivamente, se compilado com libzip ≥ 1.6.0.
ZipArchive::CM_XZ (integer)
Algoritmo XZ. Disponível a partir do PHP 7.4.3 e PECL zip 1.16.1, respectivamente, se compilado com libzip ≥ 1.6.0.
ZipArchive::ER_OK (integer)
Sem erro.
ZipArchive::ER_MULTIDISK (integer)
Arquivos Multi-disk zip não suportados.
ZipArchive::ER_RENAME (integer)
Renomeia arquivo temporário falhado.
ZipArchive::ER_CLOSE (integer)
Fecha arquivo zip falhado
ZipArchive::ER_SEEK (integer)
Procura o erro
ZipArchive::ER_READ (integer)
Lê o erro
ZipArchive::ER_WRITE (integer)
Escreve o erro
ZipArchive::ER_CRC (integer)
CRC error
ZipArchive::ER_ZIPCLOSED (integer)
Containing zip archive was closed
ZipArchive::ER_NOENT (integer)
Arquivo não encontrado.
ZipArchive::ER_EXISTS (integer)
Arquivo já existente.
ZipArchive::ER_OPEN (integer)
Não foi possível abrir o arquivo.
ZipArchive::ER_TMPOPEN (integer)
Falha ao criar arquivo temporário.
ZipArchive::ER_ZLIB (integer)
Erro da Zlib
ZipArchive::ER_MEMORY (integer)
Falha na alocação de memória.
ZipArchive::ER_CHANGED (string)
Entrada foi modificada.
ZipArchive::ER_COMPNOTSUPP (integer)
Método de compressão não suportado.
ZipArchive::ER_EOF (integer)
Prematuro EOF
ZipArchive::ER_INVAL (integer)
Argumento inválido.
ZipArchive::ER_NOZIP (integer)
Não é um arquivo zip.
ZipArchive::ER_INTERNAL (integer)
Erro interno.
ZipArchive::ER_INCONS (integer)
Arquivo Zip inconsistente.
ZipArchive::ER_REMOVE (integer)
Não foi possível remove o arquivo.
ZipArchive::ER_DELETED (integer)
Entrada foi deletada.
ZipArchive::ER_ENCRNOTSUPP (integer)
Método de criptografia não suportado. Disponível a partir do PHP 7.4.3 e PECL zip 1.16.1, respectivamente.
ZipArchive::ER_RDONLY (integer)
Arquivo somente de leitura. Disponível a partir do PHP 7.4.3 e PECL zip 1.16.1, respectivamente.
ZipArchive::ER_NOPASSWD (integer)
Nenhuma senha fornecida. Disponível a partir do PHP 7.4.3 e PECL zip 1.16.1, respectivamente.
ZipArchive::ER_WRONGPASSWD (integer)
Senha incorreta fornecida. Disponível a partir do PHP 7.4.3 e PECL zip 1.16.1, respectivamente.
ZipArchive::ZIP_ER_OPNOTSUPP (integer)
Operação não suportada. Disponível a partir do PHP 7.4.3 e PECL zip 1.16.1, respectivamente, se compilado com libzip ≥ 1.0.0.
ZipArchive::ZIP_ER_INUSE (integer)
Recurso ainda em uso. Disponível a partir do PHP 7.4.3 e PECL zip 1.16.1, respectivamente, se compilado com libzip ≥ 1.0.0.
ZipArchive::ZIP_ER_TELL (integer)
Informar erro. Disponível a partir do PHP 7.4.3 e PECL zip 1.16.1, respectivamente, se compilado com libzip ≥ 1.0.0.
ZipArchive::ZIP_ER_COMPRESSED_DATA (integer)
Dados compactados inválidos.. Disponível a partir do PHP 7.4.3 e PECL zip 1.16.1, respectivamente, Se compilado com libzip ≥ 1.6.0.
ZipArchive::ER_CANCELLED (integer)
Operação cancelada. Disponível a partir do PHP 7.4.3 e PECL zip 1.16.1, respectivamente, se compilado com libzip ≥ 1.6.0.
ZipArchive::EM_NONE (integer)
Sem criptografia, a partir do PHP 7.2.0, PECL zip 1.14.0, respectivamente, se compilado com libzip ≥ 1.2.0.
ZipArchive::EM_AES_128 (integer)
Criptografia AES 128, a partir do PHP 7.2.0, PECL zip 1.14.0, respectivamente, se compilado com libzip ≥ 1.2.0.
ZipArchive::EM_AES_192 (integer)
Criptografia AES 192, a partir do PHP 7.2.0 e PECL zip 1.14.0, respectivamente, se compilado com libzip ≥ 1.2.0.
ZipArchive::EM_AES_256 (integer)
Criptografia AES 256, a partir do PHP 7.2.0, PECL zip 1.14.0, respectivamente, se compilado com libzip ≥ 1.2.0.
ZipArchive::LIBZIP_VERSION (string)
Versão da biblioteca Zip. Disponível a partir do PHP 7.4.3 e PECL zip 1.16.0.
