Introdução

Esta extensão tem o objetivo de ajudar a analisar texto BBCode em para convertê-lo para HTML ou outra linguagem de marcação. Ela usa um parser e fornece melhor eficiência sobre as comuns implementações baseadas em expressões regulares. E além do mais, ela ajuda a ter um HTML válido por reordenar aberta / fechada tags e automaticamnete fechar tags não fechadas.

Desde 0.10.1 ele suporta argumento entre aspas com aspas simples, aspas dupla e aspas dupla com escape HTML.
