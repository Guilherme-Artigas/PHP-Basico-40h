fam_open

(PHP 5 < 5.1.0)

fam_open — Abre uma conexão para o daemon FAM

Descrição

fam_open ([ string $appname ] ) : resource

Abre uma conexão para o servidor do FAM.

Parâmetros

appname

Uma string identificando a aplicação para logar

Valor Retornado

Retorna um resource representando a conexão para o serviço FAM em sucesso ou FALSE em erros.
