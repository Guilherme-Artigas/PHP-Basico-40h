msql_close

(PHP 4, PHP 5 < 5.3.0)

msql_close — Fecha uma conexão mSQL

Descrição

msql_close ([ resource $link_identifier ] ) : int

msql_close() fecha a conexão com o banco de dados mSQL que esta associado a conexão indicada. Se o indentificador de conexão não é especificado, a última conexão aberta é assumida.

Retorna TRUE em caso de sucesso, FALSE em caso de erro.

Note que isto normalmente não é necesssário, já que conexões não persistentes são fechadas automaticamente ao final do script.

msql_close() não irá fechar conexões persistentes geradas por msql_pconnect().

Veja também: msql_connect() e msql_pconnect().
