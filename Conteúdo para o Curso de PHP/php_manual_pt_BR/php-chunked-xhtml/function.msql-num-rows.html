msql_num_rows

(PHP 4, PHP 5 < 5.3.0)

msql_num_rows — Retorna o número de registros em um resultado

Descrição

msql_num_rows ( resource $query_identifier ) : int

msql_num_rows() retorna o número de registros em um conjunto de resultados.

Parâmetros

result

O resulatdo resource que esta sendo avaliado. Este resultado vem de uma chamada a msql_query().

Valor Retornado

Returns the number of rows in the result set.
