win32_ps_list_procs

(PECL win32ps >= 1.0.1)

win32_ps_list_procs — Lista processos sendo executados

Descrição

win32_ps_list_procs ( void ) : array

Obtem estatísticas sobre todos os processos sendo executados.

Valor Retornado

Retorna FALSE em caso de falha ou um array com estatísticas dos processos como retornadas por win32_ps_stat_proc() para todos os processos sendo executados.
