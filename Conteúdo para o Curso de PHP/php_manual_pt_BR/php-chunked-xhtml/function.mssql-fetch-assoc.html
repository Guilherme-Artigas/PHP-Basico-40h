mssql_fetch_assoc

(PHP 4 >= 4.2.0, PHP 5, PECL odbtp >= 1.1.1)

mssql_fetch_assoc — Retorna um array associativo da linha atual no resultado

Descrição

mssql_fetch_assoc ( resource $result_id ) : array

Retorna um array associativo que corresponde a linha obtida e move o ponteiro de dados interno à frente. mssql_fetch_assoc() é equivalente a chamar mssql_fetch_array() com MSSQL_ASSOC para o segundo parâmetro opcional.

Parâmetros

result_id

O resource do resultado que está sendo avaliado. Este resultado vem de uma chamada a mssql_query().

Valor Retornado

Retorna um array associativo que corresponde a linha obtida, ou FALSE se não houver mais linhas.
