msql_select_db

(PHP 4, PHP 5 < 5.3.0)

msql_select_db — Seleciona um banco de dados mSQL

Descrição

msql_select_db ( string $database_name [, resource $link_identifier ] ) : bool

msql_select_db() define o banco de dados atual para a conexão com o servidor especificada. Se não for especificado um identificador de conexão, a última conexão aberta é usada. Se não houver conexão aberta, a função tentará criar uma como se a função msql_connect() fosse chamada, e ira usa-la.

Retorna TRUE em caso de sucesso, FALSE em caso de erro.

Todas as próximas chamadas a msql_query() serão feitas no banco de dados ativo.

Para compatibilidade com versões anteriore, o apelido chamado msql_selectdb() pode ser usado. Este, entretanto esta obsoleto e não é recomendado.

Veja também msql_connect(), msql_pconnect() e msql_query().
