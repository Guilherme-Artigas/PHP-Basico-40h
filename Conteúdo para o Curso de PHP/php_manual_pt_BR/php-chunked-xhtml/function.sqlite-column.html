sqlite_column

(PHP 5 < 5.4.0, PECL sqlite >= 1.0.0)

sqlite_column — Retorna uma coluna da linha atual do conjunto de resultados.

Descrição

sqlite_column ( resource $result , mixed $indice_ou_nome [, bool $decode_binary ] ) : mixed

Retorna o valor da coluna nomeada indice_ou_nome (se for uma string), ou a coluna numerada indice_ou_nome (se for um inteiro) apartir da linha atual do cunjunto de resultados result. O parâmetro decode_binary funciona da mesma maneira que a descrita na função sqlite_fetch_array().

Use esta função quando você esta interagindo com um conjunto de resultados grande com muitas colunas, ou com colunas que contém grande quantidades de dados.

Veja também sqlite_fetch_string().
