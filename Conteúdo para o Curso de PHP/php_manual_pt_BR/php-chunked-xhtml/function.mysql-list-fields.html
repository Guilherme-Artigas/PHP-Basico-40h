mysql_list_fields

(PHP 4, PHP 5)

mysql_list_fields — Lista os campos de uma tabela MySQL

Aviso

Esta função está obsoleta desde o PHP 5.4.0 e toda a extensão MySQL original foi removida in PHP 7.0.0. Alternativamente utilize as extensões MySQLi ou PDO_MySQL. Veja também MySQL: escolhendo uma API e a FAQ relacionada para mais informações Alternatives to this function include:

  • Consulta SQL: SHOW COLUMNS FROM algumatabela

Descrição

mysql_list_fields ( string $database_name , string $table_name [, resource $link_identifier = NULL ] ) : resource

Obtém informações sobre a tabela informada.

Esta função está obsoleta. É preferrível usar mysql_query() para realizar uma consulta SQL SHOW COLUMNS FROM table [LIKE 'name'].

Parâmetros

database_name

O nome do banco de dados que está sendo consultado.

table_name

O nome da tabela que esta sendo consultada.

link_identifier

A conexão MySQL. Se o link identifier não for especificado, o último link aberto por mysql_connect() é utilizado. Se uma conexão anterior não existir será tentado criar um a partir de uma chadama a mysql_connect() sem nenhum argumento. Se nenhuma conexão for encontrada ou estabelecida, um erro nível E_WARNING é gerado.

Valor Retornado

Um ponteiro de recurso resource em caso de sucesso, ou FALSE em caso de falha.

O ponteiro de resultado pode ser usado com mysql_field_flags(), mysql_field_len(), mysql_field_name() e mysql_field_type().

Exemplos

Exemplo #1 Alternativa para a obsoleta mysql_list_fields()

<?php
$result = mysql_query("SHOW COLUMNS FROM sometable");
if (!$result) {
    echo 'Could not run query: ' . mysql_error();
    exit;
}
if (mysql_num_rows($result) > 0) {
    while ($row = mysql_fetch_assoc($result)) {
        print_r($row);
    }
}
?>
